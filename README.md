# Unity-car-customization-using-AI
AI-based car colour customization system Built in Unity Engine

This project demonstrates a simple car customization system built using Unity and C#.

## Features
- 3D car model
- Car color change using scripts
- UI button interaction

## Technologies
- Unity
- C#

## How to Run
1. Open the project in Unity
2. Open the main scene
3. Press Play
4. Use the buttons to change the car color
